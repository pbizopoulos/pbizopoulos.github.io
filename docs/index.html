Summary

Paschalis Bizopoulos (in Greek: Πασχάλης Μπιζόπουλος) is an AI researcher who invented Sparsely activated networks. He also has experience in designing, developing, deploying and maintaining dockerized and web-based AI applications using portable tools and adhering to well-established standards.

Tools/Technologies

  • C (adhering to C89): for system programming
  • Docker (wherein the following technologies are used):
    • LaTeX: for document preparation
    • Node.js: for server-side programming
    • Python (adhering to a superset of PEP 8): for general programming
  • HTML/CSS/JavaScript (adhering to a superset of W3C standards): for client-side programming
  • Unix
    • POSIX Utilities (with GNU standard targets for make): for REPL and script programming
    • GNU Coreutils: for REPL programming
    • Vim: for text editing
    • Git: for version control
